What is Xenlism Wildfire ?

Xenlism is a technological graphics project to improve the overall

appearance and feeling of the end-user desktop.


It is about minimalism and realism.


More accurately, Wildfire is an icon theme for any system running x11

project clients like X.org or Wayland, like most of GNU/Linux or BSD

systems and their derivatives (Debian, Archlinux, Mageia or FreeBSD)


The graphics standards are inspired by Nokia's Meego and Apple's iOS

icons theme.


Xenlism project has been tested on Gnome,Unity,Mate and Cinnamon but there is no reason for which it wouldn't work with other detktop environment such as KDE or Xfce.


Xenlism is heavy and weighs more than 200 MB because it supports more

than 700 applications.


Wildfire icons are vectorial, which means they aren't limited by the

definition of your screen but may be slow on some systems.


Xenlism's Wildfire source code has been made with Adobe Illustrator and

is made of .ai files, which have been converted to Prototype Template

icons .svg files.


Prototype Template icons is a script to easily build color by color

scheme files.


It includes a script to convert .svg or .ai to any other icon format :

.ai→.svg, .ai→.icns, .ai→.png, .svg→.icns, .svg→.svg and .svg→.png.

Debian/Ubuntu Apt-Get Repository


Xenlism Wildfire support Debian/Ubuntu Base apt-get Repositories.

Following Command line in Terminal.

add sourceforg.net public gnupg key to the APT keys database 

		sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 2B80AC38

add repository to apt-get repo 


	sudo add-apt-repository ppa:xenatt/xenlism

add repository to apt-get repo (Debian) 


	bash <(curl -s https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/sourcelist.install)

Update System and Install Xenlism Wildfire icon theme. 


	sudo apt-get update
	sudo apt-get install xenlism-wildfire-icon-theme

Arch Linux Pacman Repository


Xenlism Wildfire support Arch Linux Base Pacman Repositories.

Following Command line in Terminal. 


		sudo nano /etc/pacman.conf

add config section to /etc/pacman.conf 


	[xenlism-arch]
	SigLevel = Optional TrustAll
	Server = http://downloads.sourceforge.net/project/xenlism-wildfire/repo/arch

Update System and Install Xenlism Wildfire icon theme And Xenlism Artwork Wallpapers. 


	sudo pacman -Syyu
	sudo pacman -S xenlism-wildfire-icon-theme

Arch Linux AUR package


Install Xenlism Wildfire With AUR


	yaourt -S xenlism-wildfire-icon-theme-git

Other Linux Distributions

Xenlism Wildfire Installations Support A most Linux and *nix System with CURl or WGET and GIT.


Install with wget and git


	bash <(wget -qO- https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/git.install)

Install with curl and git


	bash <(curl -s https://raw.githubusercontent.com/xenlism/wildfire/master/INSTALL/git.install)
